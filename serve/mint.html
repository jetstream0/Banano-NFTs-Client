What goes on behind the scenes

First, the supply block is issued (using provided seed) and max supply. Then assets (images, animations) are uploaded to IPFS using the Pinata API Key. Then, with information from prior steps is combined with description and name. Finally, that is uploaded to IPFS also, and a banano address is given. Set your rep to the given address, and any sends while you have that representative will mint and send NFTs.
